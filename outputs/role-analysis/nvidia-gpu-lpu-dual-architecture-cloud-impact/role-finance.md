## 3. 수익 모델 변화: GPU 리스 → GPU+LPU 복합 포트폴리오 마진 분석

### 현재 vs 전환 후

| 구분 | 현재 (GPU 단일) | 전환 후 (GPU+LPU) |
|------|-----------------|-------------------|
| **주요 수익원** | GPU 인스턴스 $/hr (학습·추론 동일) | GPU $/hr(학습) + LPU $/M tokens(추론) |
| **참조 가격** | CoreWeave B200 $1.02/hr, Lambda $5.74/hr | LPU $45/M tokens (NVIDIA 프리미엄) |
| **마진 구조** | GPU 리스 마진 30–50% | GPU 유지 + LPU 추가 마진 |

### 마진 분석 (LPU 추론)

| 항목 | 수치 | 비고 |
|------|------|------|
| **LPU 추론 매출** | $45/M tokens | NVIDIA/Groq 프리미엄 포지셔닝 |
| **OpenAI 대비** | 3배 | $15/M tokens — LPU는 TTFT·지연시간 프리미엄 |
| **직접 LPX 구매 시** | COGS: 전력·냉각·감가상각 | Gross margin 50–60% 가정 |
| **GroqCloud API 재판매 시** | COGS: API 호출 비용 | Gross margin 20–35% (재판매 마진) |

**결론**: 직접 LPX 구매 시 마진 우위, 재판매 시 CapEx 부담 없으나 마진 낮음. **하이브리드**(초기 재판매 → 검증 후 직접 구매)가 재무적으로 합리적.
