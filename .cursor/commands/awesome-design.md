## Awesome Design

Apply 50+ design styles, 21 palettes, 50 font pairings, and 12+ layout patterns to any UI project. Supports React, Next.js, Vue, Svelte, and more.

### Usage

```
/awesome-design plan "landing page"            # plan UI/UX approach
/awesome-design build "dashboard"              # build with best practices
/awesome-design review src/components/          # review existing UI
/awesome-design fix src/pages/Home.tsx          # fix UI issues
```

### Workflow

1. **Analyze** — Understand the UI goal and target audience
2. **Select** — Choose style, palette, typography, and layout pattern
3. **Design** — Apply design system principles: spacing, hierarchy, color
4. **Implement** — Generate or refactor code with chosen stack
5. **Review** — Verify accessibility, responsiveness, and visual polish

### Execution

Read and follow the `ui-ux-pro-max` skill (`.claude/skills/ui-ux-pro-max/SKILL.md`) for comprehensive UI/UX design intelligence across 9 stacks.
