# Pricing Strategy Design

Build a pricing strategy from first principles: pricing model analysis, WTP estimation, competitive benchmarking, and pricing experiment design.

## Usage
```
/pm-pricing SaaS project management tool moving from free to paid
/pm-pricing Should we switch from per-seat to usage-based pricing?
/pm-pricing 무료에서 유료로 전환하는 SaaS 가격 전략
```

## Workflow
1. Understand pricing context: product, current pricing, trigger, target customer, constraints.
2. Read and apply pm-product-strategy skill, invoking **pricing-strategy** and **monetization-strategy** sub-skills.
3. Evaluate applicable models: flat-rate, per-seat, usage-based, tiered, freemium, hybrid.
4. Benchmark against 3-5 competitors using web research.
5. Estimate willingness to pay (Van Westendorp if data available, or heuristic estimation).
6. Generate recommendation: pricing structure, free/trial strategy, competitive benchmark, revenue projections (conservative/expected/optimistic), migration plan, pricing experiments.
7. Save as markdown. Suggest: monetization strategy, market scan, customer communication draft, A/B test design.
