## Meeting Preparation

Prepare for your next meeting: agenda, attendees, and linked documents.

### Usage

```
/gws-meeting-prep [--calendar <calendar_id>]
```

### Workflow

1. Run `gws workflow +meeting-prep` to pull the next upcoming event
2. List attendees and their roles
3. Extract linked Drive documents from the event description
4. Summarize the preparation brief

### Examples

```
/gws-meeting-prep
/gws-meeting-prep --calendar work@company.com
```

### Execution

Read and follow the `gws-workflows` skill (`.cursor/skills/gws-workflows/SKILL.md`) for the meeting-prep workflow and other cross-service workflows (standup, weekly digest, file announce).
