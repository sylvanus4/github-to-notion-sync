## PM Product Strategy

Product strategy workflows: vision, strategy canvas, value propositions, Lean/Business Model/Startup Canvas, SWOT, PESTLE, Ansoff, Porter's Five Forces, monetization, and pricing.

### Usage

```
<sub-skill> [context]
```

### Sub-Skills

| Sub-Skill | Shorthand | What it does |
|-----------|-----------|--------------|
| product-vision | `vision` | Create inspiring, achievable product visions |
| product-strategy | `strategy` | Build 9-section Product Strategy Canvas |
| value-proposition | `value-prop` | 6-part JTBD value proposition design |
| lean-canvas | `lean` | Lean Canvas for hypothesis testing |
| business-model | `bmc` | Business Model Canvas (9 blocks) |
| startup-canvas | `startup` | Combined Strategy + Business Model for new ventures |
| swot-analysis | `swot` | SWOT analysis with recommendations |
| pestle-analysis | `pestle` | PESTLE macro environment analysis |
| porters-five-forces | `porter` | Porter's Five Forces industry analysis |
| ansoff-matrix | `ansoff` | Growth strategy mapping (Ansoff Matrix) |
| monetization-strategy | `monetize` | Propose 3-5 revenue models with validation experiments |
| pricing-strategy | `pricing` | Pricing models, WTP, price elasticity analysis |

### Execution

Read and follow the `pm-product-strategy` skill (`.cursor/skills/pm-product-strategy/SKILL.md`) for the full workflow, sub-skill selection, and error handling.

### Examples

```bash
# Build product strategy canvas
/pm-product-strategy strategy -- 9-section canvas for our B2B analytics platform

# Create product vision
/pm-product-strategy vision -- vision for our AI writing assistant

# Lean Canvas for new venture
/pm-product-strategy lean -- hypothesis canvas for marketplace MVP

# SWOT + PESTLE macro analysis
/pm-product-strategy swot -- strategic assessment of our EdTech position

# Pricing strategy
/pm-product-strategy pricing -- compare freemium vs usage-based for our API
```
