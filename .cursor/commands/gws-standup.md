## Daily Standup Report

Generate a daily standup summary from Google Calendar and Tasks.

### Usage

```
/gws-standup
```

### Workflow

1. Run `gws workflow +standup-report` to get today's meetings and open tasks
2. Present the standup summary in a readable format
3. Optionally format as table with `--format table`

### Execution

Read and follow the `gws-workflows` skill (`.cursor/skills/gws-workflows/SKILL.md`) for the standup-report workflow and other cross-service workflows (meeting prep, weekly digest, email-to-task).
