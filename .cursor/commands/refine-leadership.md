# Refine Leadership — 리더십 글 변환

## Skill Reference

Read and follow the skill at `.cursor/skills/standalone/leadership-writing-refiner/SKILL.md`.

## Usage

Pass the text to be transformed as the argument. The skill will:

1. Classify the text type and audience
2. Diagnose communication anti-patterns (judgment leakage, speed-over-empathy, missing expectations, psychological distance)
3. Apply 4 transformation lenses (delivery refinement, context-first reframing, expectation clarity, growth guide insertion)
4. Compose the refined Korean output
5. Polish via `sentence-polisher`

All output is in **Korean**.
