# Macro-Environment Analysis

Run multiple strategic analysis frameworks to understand the competitive and macro environment. Combines SWOT, PESTLE, Porter's Five Forces, and Ansoff Matrix into a single strategic overview.

## Usage
```
/pm-market-scan EdTech market for corporate learning
/pm-market-scan Our fintech product — preparing for board strategy review
/pm-market-scan 기업 학습용 EdTech 시장 분석
```

## Workflow
1. Understand analysis target, purpose, and current position.
2. Read and apply pm-product-strategy skill, invoking four sub-skills sequentially:
   - **swot-analysis**: Internal strengths/weaknesses, external opportunities/threats
   - **pestle-analysis**: Political, Economic, Social, Technological, Legal, Environmental
   - **porters-five-forces**: Rivalry, supplier/buyer power, substitutes, new entrants
   - **ansoff-matrix**: Penetration, market development, product development, diversification
3. Cross-reference results to identify convergent signals, strategic imperatives, key risks, and growth opportunities.
4. Generate comprehensive report with executive summary. Save as markdown.
5. Suggest: build product strategy, analyze specific competitors, design pricing strategy.
