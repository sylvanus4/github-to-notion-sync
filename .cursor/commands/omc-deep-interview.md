## Deep Interview

Socratic deep interview with mathematical ambiguity scoring. Asks one targeted question at a time, scores clarity across weighted dimensions, and gates on ambiguity threshold (default 20%) before proceeding to execution.

### Usage

```
<idea or feature description>
```

### Execution

Read and follow the `omc-deep-interview` skill (`.cursor/skills/omc-deep-interview/SKILL.md`) for the full workflow.

### Examples

```bash
# Interview for a vague idea
/omc-deep-interview I want to build a task management system

# Clarify requirements before a feature
/omc-deep-interview Add authentication to the app

# Socratic interview for a complex feature
/omc-deep-interview We need a real-time notification system for stock alerts
```
