## Algorithmic Art

Create generative art using p5.js with seeded randomness, flow fields, particle systems, and interactive parameters.

### Usage

```
/algorithmic-art "flow field with warm colors"
/algorithmic-art --style particles "cosmic dust"
/algorithmic-art --interactive "sound reactive"
```

### Workflow

1. **Concept** — Interpret the artistic vision and choose algorithms
2. **Generate** — Write p5.js sketch with seeded randomness for reproducibility
3. **Iterate** — Adjust parameters: color, density, speed, behavior
4. **Export** — Output as HTML file with embedded p5.js

### Execution

Read and follow the `anthropic-algorithmic-art` skill (`.cursor/skills/anthropic/anthropic-algorithmic-art/SKILL.md`) for creating algorithmic art with p5.js, seeded randomness, and interactive parameter exploration.
