# Value Proposition Design

Design a clear, compelling value proposition using the 6-part JTBD template. Starts from the customer and works toward practical outcomes.

## Usage
```
/pm-value-proposition AI writing tool for non-native English speakers
/pm-value-proposition [upload pitch deck or competitive analysis]
/pm-value-proposition 비영어권 사용자를 위한 AI 글쓰기 도구 가치 제안
```

## Workflow
1. Understand product and market context.
2. Read and apply pm-product-strategy skill, invoking **value-proposition** sub-skill.
3. Generate 6-part template: Who, Why, What Before, How, What After, Alternatives.
4. Create value proposition statements for marketing, sales, and onboarding.
5. If multiple segments, create separate VPs for each.
6. Save as markdown. Suggest: compare with competitors via Value Curve, build full strategy, create Lean Canvas.
