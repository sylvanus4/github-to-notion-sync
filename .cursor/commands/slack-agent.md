## Slack Agent

Build a Slack bot or AI agent using Bolt for JavaScript. Deploy to any platform (Docker, K8s, serverless, PaaS).

### Usage

```
/slack-agent              # Auto-detect project state and route to appropriate phase
/slack-agent new          # Start new project wizard from scratch
/slack-agent configure    # Configure Slack app and environment for existing project
/slack-agent deploy       # Deploy to production (choose platform)
/slack-agent test         # Set up testing infrastructure
```

### Execution

Read and follow the `slack-agent` skill (`.cursor/skills/slack-agent/SKILL.md`).
