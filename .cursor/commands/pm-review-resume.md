# PM Review Resume

Comprehensive PM resume review against 10 best practices — structure, impact metrics, keywords, and actionable feedback. References pm-toolkit skill, review-resume sub-skill. Score each practice 1-10, provide top 3 improvements with before/after examples.

## Usage

```
/pm-review-resume Review my resume for Senior PM roles
/pm-review-resume 내 PM 이력서 리뷰해줘, 10가지 관점으로
```

## Workflow

### Step 1: Load skill and reference

Read the `pm-toolkit` skill (`.cursor/skills/pm-toolkit/SKILL.md`) and `references/review-resume.md`.

### Step 2: Obtain resume

Request or use provided:

- Resume text (pasted or file path)
- Optional: Job posting for role-specific feedback

### Step 3: Evaluate against 10 best practices

Score each practice 1–10 with justification:

1. Clear structure and formatting
2. Quantified impact (metrics, outcomes)
3. Action verbs and strong opening lines
4. Relevance to PM skills (discovery, prioritization, execution)
5. Keyword alignment with target roles
6. Concision (no fluff, appropriate length)
7. Achievement focus vs responsibility listing
8. Consistency (dates, formatting, voice)
9. Tailoring to audience (hiring manager, recruiter)
10. Notable differentiators and story

Include 1–2 representative quotes per practice where relevant.

### Step 4: Top 3 improvements

Select the highest-impact improvements and provide:

- Before: current wording
- After: revised wording (XYZ+S formula: X achievement, Y method, Z result, S scale)
- Rationale for each change

### Step 5: Output

Deliver:

1. Brief intro (overall assessment)
2. Detailed feedback per practice with score and quotes
3. Top 3 improvements with before/after
4. Conclusion and next steps

## Notes

- Tailor scoring if a job posting is provided (emphasize keyword/match).
- XYZ+S: “Increased activation by 15% by simplifying onboarding, driving $2M incremental ARR.”
- Keep tone professional; avoid overly generic advice.
