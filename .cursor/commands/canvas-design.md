## Canvas Design

Create visual art, posters, social media graphics, and covers as PNG/PDF from text prompts.

### Usage

```
/canvas-design "conference poster for AI Summit 2026"
/canvas-design --format poster "product launch"
/canvas-design --format social "Twitter header"
```

### Workflow

1. **Brief** — Clarify visual intent: audience, medium, dimensions, mood
2. **Compose** — Design layout, typography, and visual hierarchy
3. **Create** — Generate the visual as a self-contained design
4. **Export** — Output as PNG or PDF

### Execution

Read and follow the `anthropic-canvas-design` skill (`.cursor/skills/anthropic/anthropic-canvas-design/SKILL.md`) for creating static visual art and design pieces.

### Examples

Create a conference poster:
```
/canvas-design "minimalist poster for cloud computing workshop"
```

Create social media graphics:
```
/canvas-design --format social "LinkedIn banner for AI platform launch"
```
