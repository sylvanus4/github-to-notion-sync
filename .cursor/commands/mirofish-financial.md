## MiroFish Financial Simulation

Simulate financial market reactions using MiroFish multi-agent engine. Agents include retail investors, hedge fund managers, analysts, bond traders, and crypto traders.

### Usage

`<financial scenario description>`

### Execution

Read and follow the `mirofish-financial-sim` skill for the specialized financial workflow.

### Examples

```bash
/mirofish-financial -- Simulate market reaction to NVIDIA 10:1 stock split announcement
/mirofish-financial -- What happens if the Fed announces a surprise 50bp rate cut?
/mirofish-financial -- Predict 30-day S&P 500 trajectory after CPI prints at 5.2% vs 4.1% consensus
/mirofish-financial -- How will crypto markets react to SEC approving spot Ethereum ETF?
/mirofish-financial -- Simulate impact of China announcing 500B RMB stimulus package
```
