# Business Model Exploration

Build and analyze business models using four complementary frameworks. Choose one or run all for a complete picture.

## Usage
```
/pm-business-model lean Marketplace connecting freelance PMs with startups
/pm-business-model full Enterprise analytics platform
/pm-business-model startup AI writing tool for non-native speakers
/pm-business-model all SaaS onboarding tool
/pm-business-model 프리랜서 PM 마켓플레이스 린 캔버스
```

## Modes

### Lean Canvas (lean)
Apply pm-product-strategy skill, invoking **lean-canvas** sub-skill. Best for: early-stage ideas, startups, new product lines.

### Full BMC (full)
Apply pm-product-strategy skill, invoking **business-model** sub-skill. Best for: mature products, strategic planning, investor materials.

### Startup Canvas (startup)
Apply pm-product-strategy skill, invoking **startup-canvas** sub-skill. Best for: new products needing strategy + business model in one artifact. Recommended starting point for new products.

### Value Proposition (value-prop)
Apply pm-product-strategy skill, invoking **value-proposition** sub-skill. Best for: messaging refinement, PMF analysis.

### All
Run all four and add a cross-framework synthesis section.

## Workflow
1. Understand context (product/idea, stage, existing model, target customer).
2. Generate selected framework(s).
3. Save as markdown. Suggest follow-ups: stress-test with SWOT/PESTLE, design pricing, build strategy canvas.
