# Team OKR Planning

Generate systematic OKRs that connect team work to company strategy. Presents 3 OKR sets with qualitative objectives and quantitative key results.

## Usage
```
/pm-plan-okrs Growth team Q2 — company goal is 50% ARR increase
/pm-plan-okrs Onboarding squad aligned to "improve activation rate"
/pm-plan-okrs 그로스팀 Q2 OKR — 회사 목표 ARR 50% 증가
```

## Workflow
1. Gather context: team, period, company-level goals, last quarter performance, constraints.
2. Read and apply pm-execution skill, invoking **brainstorm-okrs** sub-skill.
3. Generate 3 OKR sets (Objective + 3-5 Key Results each). Objectives: qualitative, inspiring. KRs: quantitative, time-bound, with clear owners.
4. Quality check: inspiring objectives, measurable KRs, stretch targets (70% = well-calibrated), 3-5 KRs per objective, no gameable KRs.
5. Present with alignment map, scoring guide (0.0-1.0), and check-in cadence.
6. Suggest: adjust targets, build metrics dashboard, write stakeholder update.
