## Frontend Design

Generate production-grade frontend interfaces with distinctive, polished React + Tailwind components that avoid generic AI aesthetics.

### Usage

```
/frontend-design "user settings page"          # design from description
/frontend-design --style glassmorphism         # specific visual style
/frontend-design --responsive                  # mobile-first responsive
```

### Workflow

1. **Understand** — Clarify requirements: layout, interactions, data, constraints
2. **Design** — Generate component hierarchy and visual direction
3. **Implement** — Build React + Tailwind components with proper state management
4. **Polish** — Add micro-animations, hover states, and responsive behavior
5. **Output** — Production-ready component code with proper types

### Execution

Read and follow the `anthropic-frontend-design` skill (`.cursor/skills/anthropic/anthropic-frontend-design/SKILL.md`) for distinctive, production-grade frontend interface creation.

### Examples

Design a settings page:
```
/frontend-design "user settings page with theme toggle and notification preferences"
```

Design with a specific style:
```
/frontend-design --style "dark minimalist" "analytics dashboard"
```
