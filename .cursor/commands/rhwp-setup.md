# /rhwp-setup

Install and configure the complete rhwp HWP/HWPX document toolkit.

## Instruction

Read and follow the `rhwp-setup` skill at `.cursor/skills/standalone/rhwp-setup/SKILL.md`.

Execute all steps sequentially:
1. Check prerequisites (Rust, Node.js, Docker)
2. Clone or update the rhwp repository
3. Build the CLI from source
4. Install the VS Code extension
5. Install npm packages (optional)
6. Run diagnostic health check
7. Run smoke test

Report the final component status table.
