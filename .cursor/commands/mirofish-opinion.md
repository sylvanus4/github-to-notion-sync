## MiroFish Opinion Simulation

Simulate public opinion dynamics across Twitter and Reddit using MiroFish multi-agent engine. Agents include consumers, journalists, regulators, influencers, and competing brands.

### Usage

`<public opinion scenario description>`

### Execution

Read and follow the `mirofish-opinion-sim` skill for the specialized opinion simulation workflow.

### Examples

```bash
/mirofish-opinion -- Simulate public reaction if our CEO makes a controversial political statement
/mirofish-opinion -- Predict 7-day social media trajectory after a product recall announcement
/mirofish-opinion -- How would Chinese social media react to a new data privacy regulation?
/mirofish-opinion -- Simulate university campus response to a controversial admissions policy
/mirofish-opinion -- What if a viral TikTok exposes a manufacturing defect in our product?
```
