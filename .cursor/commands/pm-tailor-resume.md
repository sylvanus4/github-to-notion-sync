# PM Tailor Resume

Optimize a PM resume for a specific job posting — keyword alignment, experience reframing, and strategic optimization. References pm-toolkit skill, review-resume sub-skill. Analyze JD, perform keyword gap analysis, rewrite bullets with XYZ+S formula.

## Usage

```
/pm-tailor-resume Tailor my resume to this job posting [paste JD]
/pm-tailor-resume 이 공고에 맞게 이력서 최적화해줘
```

## Workflow

### Step 1: Load skill and reference

Read the `pm-toolkit` skill (`.cursor/skills/pm-toolkit/SKILL.md`) and `references/review-resume.md`.

### Step 2: Extract job posting requirements

From the job description (JD):

- Required skills, tools, domain experience
- Preferred qualifications
- Key phrases and terminology
- Company/role priorities (e.g., growth, B2B, platform)

### Step 3: Keyword gap analysis

Compare resume vs JD:

- **Present and strong** — Keywords well represented
- **Present but weak** — Could be emphasized or reframed
- **Missing** — Gaps; suggest where to add (existing experience reframe vs new bullet)
- **Overused** — Generic terms to replace with JD language

Produce a gap table.

### Step 4: Rewrite bullets with XYZ+S

For 3–5 high-impact bullets:

- Apply XYZ+S: [Achievement] by [Method], [Result] at [Scale]
- Inject JD keywords naturally
- Align language with role level and company tone
- Preserve authenticity; avoid fabrication

Provide before/after for each rewritten bullet.

### Step 5: Strategic adjustments

Recommend:

- Section ordering (e.g., move relevant experience up)
- Summary/headline tweaks for this role
- What to emphasize in cover letter
- Optional additions (certifications, projects) if gaps remain

### Step 6: Output

Deliver:

1. JD analysis summary
2. Keyword gap table
3. Rewritten bullets (before/after)
4. Strategic recommendations
5. Final tailored resume (or marked-up version)

## Notes

- Never invent experience; reframe existing work to match JD.
- ATS keyword matching matters; use exact phrases where appropriate.
- Tailor 3–5 bullets deeply rather than surface-editing the whole resume.
