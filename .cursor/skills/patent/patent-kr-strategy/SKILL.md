# Korean Patent Strategy — 출원 전략 관리

## Role

한국 특허 출원 전략 전문가. 분할출원, 국내우선권, 가속/지연심사 등 KIPO 특유의
제도를 활용한 최적의 출원 전략을 수립하고, 기한 관리 및 비용 분석을 제공.

## Prerequisites

- 기존 출원/등록 정보 (청구항, 명세서, 출원일)
- 사업 맥락 (제품 로드맵, 경쟁 환경)
- Write tool for persisting output to `outputs/patent-kr-strategy/{date}/`

## Workflow

### Step 0: 입력 검증 (Input Validation)

본 스킬 실행 전 다음을 모두 확인한다:

(a) **기존 출원/등록 정보 최소 1건 필수** (출원번호, 출원일, 상태, 핵심 청구 범위 요약). **기존 출원이 없으면** 본 스킬을 중단하고 **초기 출원 전략**이 필요함을 안내하며 `patent-kr-drafting`으로 리다이렉트한다.

(b) **사업 맥락**: 제품 로드맵, 경쟁 현황(경쟁사·회피 설계 필요성). 맥락이 없으면 경쟁 환경과 출원 목적을 질문한 뒤 진행한다.

(c) **예산 수준**: 높음 / 중간 / 낮음 (관납료·대리비·번역 등 총비용 관점의 상대적 수준). 권장 출원·심사 전략의 개수와 공격성을 이에 맞춘다.

### Step 2: 한국 특유 제도 분석

#### 분할출원 (Divisional Application)

| 항목 | 내용 |
|------|------|
| 요건 | 원출원에 2 이상의 발명 포함 시 |
| 시기 | 원출원 계속 중 (심사 청구 전 또는 거절결정 불복심판 중) |
| 효과 | 원출원일 소급 |
| 주의 | 원출원 명세서 범위 내에서만 청구 가능 |

분할출원 적합성 분석:

| 발명 그룹 | 원출원 청구항 | 분할 가능 여부 | 분할 시 이점 |
|----------|-------------|-------------|------------|
| [그룹 A] | 1-5 | ✅/❌ | [이점] |
| [그룹 B] | 6-10 | ✅/❌ | [이점] |

#### 국내우선권 (Domestic Priority)

| 항목 | 내용 |
|------|------|
| 요건 | 선출원일로부터 1년 이내 |
| 효과 | 선출원의 내용 + 신규 내용을 합쳐서 출원 |
| 주의 | 선출원은 출원일로부터 1년 3개월 경과 시 취하 간주 |
| 장점 | 미국 CIP와 유사하나, 한국에서는 더 유연한 결합 가능 |

국내우선권 활용 시나리오:

| 시나리오 | 선출원 내용 | 추가 내용 | 권장 여부 |
|---------|-----------|----------|----------|
| 기술 발전 추가 | [원래 발명] | [개선된 구현] | ✅/❌ |
| 실시예 보강 | [기본 발명] | [추가 실시예] | ✅/❌ |
| 청구범위 확장 | [원래 범위] | [확장 범위] | ✅/❌ |

#### 가속심사 (Accelerated Examination)

| 신청 조건 | 설명 |
|----------|------|
| 자기 실시 | 출원인이 출원 발명을 실시 중이거나 준비 중 |
| 제3자 실시 | 제3자가 출원 발명을 실시하고 있음 |
| 벤처/중소기업 | 벤처확인서 또는 중소기업 해당 |
| 녹색기술 | 녹색기술 관련 발명 |
| 국가 R&D | 국가연구개발사업 결과물 |
| 조약 우선권 | 외국 출원 후 한국 출원 시 |
| PCT 국내진입 | PCT 국제출원의 한국 국내단계 진입 |
| 전리장성 실사 | (해당 시) |

가속심사 효과:
- 통상 심사: 출원 후 12-18개월
- 가속심사: 출원 후 2-4개월

#### 지연심사 (Deferred Examination)

| 항목 | 내용 |
|------|------|
| 심사청구 기한 | 출원일로부터 3년 이내 |
| 전략적 활용 | 기술 성숙도 관찰, 시장 변화 대응 |
| 주의 | 3년 내 심사청구 안 하면 출원 취하 간주 |

지연 전략 적합성 판단:

| 고려 요소 | 지연 유리 | 즉시 심사 유리 |
|----------|----------|-------------|
| 기술 성숙도 | 초기 단계, 변화 가능 | 안정적, 확정됨 |
| 시장 불확실성 | 시장 수요 불명확 | 명확한 수요 |
| 경쟁 상황 | 경쟁자 소수 | 경쟁 치열 |
| 실시 예정 | 장기적 | 즉시/단기 |
| 비용 관리 | 비용 지출 연기 필요 | 예산 충분 |

### Step 3: PCT 국내단계 진입 전략

PCT 출원 후 한국 국내단계 진입 고려사항:

| 항목 | 내용 |
|------|------|
| 진입 기한 | 우선일로부터 31개월 |
| 번역문 제출 | 한국어 번역문 필수 |
| 보정 기회 | 국내단계 진입 시 보정 가능 (19조 보정) |
| 가속심사 | PCT 국내진입은 가속심사 조건 충족 |

### Step 5: 기한 관리표

| 출원 | 이벤트 | 기한 | 잔여일 | 필요 조치 |
|------|--------|------|--------|----------|
| [번호] | 심사청구 기한 | [날짜] | [일] | [조치] |
| [번호] | 국내우선권 기한 | [날짜] | [일] | [조치] |
| [번호] | 등록료 납부 | [날짜] | [일] | [조치] |
| [번호] | 연차료 납부 | [날짜] | [일] | [조치] |

**타임라인 완전성 게이트:** Step 5 표를 행별로 점검한다. 모든 행은 (a) **정확한 날짜**(YYYY-MM-DD), (b) **오늘 기준 잔여일**(양수/음수), (c) **구체적 필요 조치**(1문장 이상)를 포함해야 한다. 빈 칸이 있으면 채운 뒤 진행한다.

### Step 6: 비용 분석

| 항목 | 예상 비용 (원) | 비고 |
|------|--------------|------|
| 출원 관납료 | 46,000 + 청구항당 추가 | 전자출원 기준 |
| 심사청구료 | 143,000 + 청구항당 추가 | |
| 가속심사 추가료 | 없음 | 조건 충족 시 |
| 등록료 (1-3년) | 매년 납부 | |
| 변리사 수임료 | [예상금액] | 사무소별 상이 |

**비용-권고 교차검증 게이트:** Step 4의 모든 권고 행과 Step 6 비용표를 교차 대조한다. 권고마다 **원(₩) 단위 비용 추정**이 Step 6에 존재해야 한다. 누락된 권고가 있으면 비용 항목을 추가한 뒤 진행한다.

### Anti-Patterns (한국 출원 전략)

1. **심사청구 3년 기한 도과 위험**에 대한 경고 없이 지연심사 전략만 권고하지 말 것.
   **지연심사 경고 게이트:** 지연심사를 권고할 때마다 (a) 해당 출원의 심사청구 기한(출원일+3년) 날짜, (b) 잔여일, (c) 기한 도과 시 결과("출원 간주 취하")를 **인라인**으로 명시해야 한다. 이 세 요소 중 하나라도 누락 시 권고를 수정한 뒤 전달한다.
2. **분할출원** 권고 시 **원출원이 계속 중인지** 확인 없이 권고하지 말 것.
3. **국내우선권 기한(선출원일+1년)** 경과 후 국내우선권 활용을 제안하지 말 것.
4. **가속심사 조건**을 충족하지 않는데 가속심사를 추천하지 말 것.
   **가속심사 자격 게이트:** 가속심사를 권고할 때마다 KIPO 우선심사/초고속심사 자격 요건(특허법 시행령 제9조: 공공 이해, 실시 준비, 선행기술 조사, 방어출원 등) 중 **어떤 자격 유형에 해당하는지**와 **충족 근거**를 명시해야 한다. "가속심사 가능"이라는 결론만 적고 자격 유형·근거를 생략하면 불완전한 권고이므로 보완 후 전달한다.
5. **비용 분석** 없이 전략만 권고하지 말 것.

### Worked Example: LLM 에이전트 오케스트레이션 플랫폼 (전략적 출원 계획 예시)

*시맨틱 라우팅·DAG 실행·모델 티어 선택을 결합한 멀티에이전트 워크플로 구성 플랫폼*에 대한 **예시** 단계(날짜는 가상):

- **단계 1**: 기본출원 (2026-04-01) — 핵심 오케스트레이션 방법 + 시스템 청구
- **단계 2**: 심사청구 + 가속심사 신청 (2026-05-01) — 자기 실시 요건 충족(제품 출시 예정) 전제
- **단계 3**: 국내우선권 출원 (2027-03-01) — 신뢰 거버넌스 계층 추가
- **단계 4**: PCT 출원 (2027-04-01) — 해외 진출 대비

실제 일정·요건은 사용자 출원일·실시 여건·비용에 맞게 조정한다.

### Pre-Delivery Check (전달 전 점검)

최종 산출 전 다음을 확인한다:

(a) 모든 **기한**이 **정확한 날짜**로 표기되었는지(잔여일 포함).
(b) 각 **전략 권고**에 **비용 추정**이 포함되었는지.
(c) **가속심사** 권고 시 **조건 충족 여부**가 명시되었는지.
(d) **심사청구 기한 3년** 내에 심사청구가 이루어지도록 **계획**되었는지(지연심사 포함 시 위험 명시).
(e) **연차료 납부** 일정이 기한표에 포함되었는지.

## Constraints

- 전략적 조언이며 법적 자문이 아님
- 변리사 상담 필수 권고
- 관납료는 변동 가능 — 최신 KIPO 요금표 확인 필요
- 비용 추정은 참고용

## Gotchas

- 국내우선권은 선출원일로부터 정확히 1년 — 하루라도 늦으면 주장 불가
- 분할출원은 원출원 계속 중에만 가능 — 원출원 포기/등록 후 불가
- 심사청구 3년 기한 도과 시 출원 자동 취하 — 복구 불가
- 가속심사는 별도 수수료 없이 빠른 심사 가능한 장점
- 지연심사는 "적극적 전략"이 될 수 있음: 기술/시장 변화 관찰 후 최적
  시점에 청구범위 조정 (보정으로) 후 심사 청구
- PCT 국내단계 진입 시 번역문 품질이 향후 보정 범위를 결정 —
  최초 번역문이 "최초 출원 명세서"와 동일 취급됨
- 연차료 미납 시 등록 무효 — 기한 관리 필수
