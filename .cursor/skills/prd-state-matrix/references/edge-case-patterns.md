# Edge Case Pattern Library

기능 분석 시 체크해야 할 Edge Case 패턴 라이브러리. 카테고리별로 정리.

## Network & API

- [ ] 네트워크 끊김 (오프라인 상태)
- [ ] API 응답 지연 (타임아웃)
- [ ] API 서버 다운 (5xx)
- [ ] 부분 응답 (일부 필드만 반환)
- [ ] 중복 요청 (따닫 클릭)
- [ ] 동시 요청 충돌 (race condition)
- [ ] 요청 중 화면 이탈

## Input & Validation

- [ ] 빈 입력 (공백만)
- [ ] 최소/최대 길이 초과
- [ ] 특수문자 입력 (SQL injection, XSS)
- [ ] 복사-붙여넣기 (서식 포함)
- [ ] 자동완성 값
- [ ] IME 조합 중 (한글 입력 중간)
- [ ] 숫자 필드에 문자 입력

## Auth & Permission

- [ ] 토큰 만료 중 작업
- [ ] 권한 변경 중 작업 (실시간)
- [ ] 멀티 디바이스 동시 로그인
- [ ] 세션 타임아웃
- [ ] 미인증 상태에서 딥링크 접근

## Data

- [ ] 데이터 0건 (빈 상태)
- [ ] 대량 데이터 (1000+ 건)
- [ ] 페이지네이션 경계
- [ ] 정렬/필터 조합
- [ ] 삭제된 데이터 참조
- [ ] 동시 편집 (concurrency)

## UI & Platform

- [ ] 긴 텍스트 오버플로
- [ ] 이미지 로드 실패
- [ ] 다국어 텍스트 길이 차이
- [ ] 가로/세로 모드 전환
- [ ] 다크모드/라이트모드
- [ ] 접근성 (스크린리더, 키보드)
- [ ] 브라우저 뒤로가기/앞으로가기

## Business Logic

- [ ] 경계값 (0, 1, max, max+1)
- [ ] 시간대 차이 (UTC vs 로컬)
- [ ] 할인/쿠폰 중복 적용
- [ ] 재고 부족 (실시간 변동)
- [ ] 환불/취소 후 재구매
- [ ] 구독 만료 후 접근
- [ ] 프로모션 기간 종료 시점
