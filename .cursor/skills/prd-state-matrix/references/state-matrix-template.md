### 상태 전이 다이어그램

```mermaid
stateDiagram-v2
    [*] --> Initial
    Initial --> Loading: [트리거]
    Loading --> Success: [조건]
    Loading --> Error: [에러 조건]
    Error --> Loading: [재시도]
    Success --> [*]
```

|---|------|----------|---------|------------|----------|------|
| 1 | 초기 (Initial) | 화면 최초 진입 | [기본 UI] | [가능한 액션] | [다음 상태] | |
| 2 | 로딩 (Loading) | [트리거 액션] | 스피너/스켈레톤 | 취소 (가능 여부) | 성공/실패 | |
| 3 | 성공 (Success) | API 응답 성공 | [성공 UI] | [후속 액션] | [다음 상태] | |
| 4 | 실패 (Error) | API 에러/타임아웃 | 에러 메시지 | 재시도/돌아가기 | 로딩/초기 | |
