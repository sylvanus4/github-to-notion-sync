# Collection Naming Convention

## Monthly Collection

**Format:** `AI Trends {YYYY-MM}`
**Example:** `AI Trends 2026-03`

**Description template:**
```
Auto-curated trending papers and models from Hugging Face Hub.
Generated by hf-trending-intelligence pipeline.
Period: {YYYY-MM-DD} to {YYYY-MM-DD}
```

## Item Notes

When adding items to the collection:

**Papers:**
```
Score: {SCORE} — {LABEL} | {DATE} | ↑{UPVOTES} | {MODEL_COUNT} models
```

**Models (if associated with HOT paper):**
```
Associated with "{PAPER_TITLE}" | {DOWNLOADS} downloads | Added {DATE}
```

## Lifecycle

- Collections are created on the first run of each month
- Items accumulate throughout the month
- At month end, the collection serves as a monthly research digest
- Collections are never deleted (permanent archive)
