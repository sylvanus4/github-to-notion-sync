# 제품 상세 페이지 빌더 (Product Detail Page Builder)

> Source: modu-ai/cowork-plugins moai-marketing campaign-planner/references/product-detail

전자상거래 플랫폼의 제품 상세 페이지를 생성합니다.
**고객 신뢰도 향상**, **전환율 극대화**, **SEO 최적화**를 모두 만족하는 페이지 구조를 제공합니다.

---

## 페이지 레이아웃

```
+----------------------------------+
|        Header (네비게이션)         |
+----------------------------------+
|  이미지 갤러리  |  제품 기본 정보   |
|  (메인+썸네일)  |  가격/평점/재고   |
|                |  구매 버튼        |
+-----------------+-----------------+
|        상세 설명 (스크롤)          |
|  - 핵심 혜택 3가지               |
|  - 기능 상세 목록                 |
|  - 사용 시나리오                  |
+---------------------------------+
|        비교 테이블                |
|  (vs 경쟁 제품 or 모델별)        |
+---------------------------------+
|        소셜 프루프                |
|  - 구매 후기, 평점 분포           |
|  - 구매 건수, 인증 리뷰           |
+---------------------------------+
|        FAQ / Q&A                 |
+---------------------------------+
|        관련 상품 추천             |
+---------------------------------+
|        하단 CTA (모바일 sticky)   |
+---------------------------------+
```

## 섹션별 가이드

### 1. Header

- 브랜드 로고 + 검색바 + 장바구니
- 브레드크럼: 홈 > 카테고리 > 서브카테고리 > 상품명
- 모바일: 햄버거 메뉴 + 고정 하단 바

### 2. 이미지 갤러리

- **메인 이미지**: 정사각형 또는 4:3, 최소 1000x1000px
- **썸네일**: 5-8장, 클릭 시 메인 전환
- **기능**: 확대(줌), 360도 회전(선택), 동영상 슬롯 1개
- **Smartstore**: 860px 폭 기준, PNG/JPG
- **Coupang**: 850x850px, 흰색 배경 권장

### 3. 제품 기본 정보

```
[브랜드명]
제품명 (키워드 포함, Smartstore 50자 / Coupang 100자)
★★★★☆ 4.8 (1,234개 리뷰) | 누적 판매 5,000+

정가: ~~59,000원~~
할인가: 49,000원 (17% 할인)
적립: 490P (1%)

배송: 무료배송 | 내일 도착
옵션: [색상 선택] [사이즈 선택]
수량: [-] 1 [+]

[장바구니 담기]  [바로 구매]
```

### 4. 상세 설명

**구조 원칙**:
- 핵심 혜택 3가지를 최상단에 아이콘+텍스트로 배치
- 이미지:텍스트 비율 = 7:3 (모바일 최적)
- 스크롤 길이: 스마트스토어 기준 5000-8000px 권장
- 각 섹션 사이 구분선 또는 배경색 변경

**필수 포함 요소**:
1. 제품 사용 전/후 비교 (비주얼)
2. 성분/소재/스펙 상세표
3. 인증/수상 마크 (KC인증, 특허, ISO 등)
4. 제조사/원산지 정보
5. 교환/반품/AS 안내

### 5. 비교 테이블

| 항목 | 이 제품 | 경쟁 A | 경쟁 B |
|------|---------|--------|--------|
| 가격 | ✅ 49,000 | 55,000 | 52,000 |
| 무료배송 | ✅ | ❌ | ✅ |
| 리뷰 수 | ✅ 1,234 | 567 | 890 |
| 보증기간 | ✅ 2년 | 1년 | 1년 |

### 6. 소셜 프루프

- 평점 분포 막대그래프 (5점~1점)
- 포토/동영상 리뷰 우선 노출
- "베스트 리뷰" 3-5개 하이라이트
- 리뷰 필터: 최신순, 평점순, 포토만
- 구매 인증 배지 표시

### 7. FAQ

- 최소 5개 질문, 아코디언 UI
- 배송, 교환/반품, 사이즈, 사용법, 호환성 필수
- 스키마 마크업 (FAQPage) 적용으로 SEO 강화

## 플랫폼별 최적화 체크리스트

### Naver Smartstore

- [ ] 상품명에 핵심 키워드 앞배치 (50자 이내)
- [ ] 카테고리 태그 정확히 매칭
- [ ] 상품 속성 100% 입력 (색상, 사이즈, 소재 등)
- [ ] 네이버 쇼핑 검색 태그 최적화
- [ ] 스토어찜 유도 배너 삽입
- [ ] 톡톡 문의 버튼 활성화
- [ ] 리뷰 작성 유도 메시지 설정

### Coupang

- [ ] 상품명에 카테고리 키워드 포함 (100자 이내)
- [ ] A+ 콘텐츠 섹션 활용
- [ ] 로켓배송/로켓그로스 배지 표시
- [ ] 묶음배송 설정
- [ ] 쿠팡 광고 키워드 연동
- [ ] 고객 Q&A 응답률 유지 (24시간 이내)

## 모바일 최적화 필수사항

- 이미지 로딩: lazy load + WebP 변환
- 폰트 크기: 본문 14px 이상, 가격 20px 이상
- CTA 버튼: 화면 하단 고정 (sticky), 높이 48px 이상
- 터치 타겟: 44x44px 이상
- 스크롤 깊이: 핵심 정보 첫 2스크롤 이내
