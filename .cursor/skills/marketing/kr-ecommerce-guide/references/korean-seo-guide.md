# 한국 SEO 가이드 (Korean SEO Guide)

> Source: modu-ai/cowork-plugins moai-marketing seo-audit + campaign-planner

네이버, 구글, AI 검색(GEO) 통합 SEO 전략 레퍼런스.

---

## 1. 네이버 C-Rank (Creator Rank)

네이버가 콘텐츠 생산자(블로거, 스토어)의 신뢰도를 평가하는 알고리즘.

### 평가 요소

| 요소 | 비중(추정) | 설명 |
|------|-----------|------|
| 주제 집중도 | ★★★★★ | 한 분야에 꾸준히 포스팅하는 전문성 |
| 콘텐츠 품질 | ★★★★☆ | 글 길이, 이미지 수, 체류시간 |
| 활동 지속성 | ★★★★☆ | 주 3회 이상 꾸준한 포스팅 |
| 사용자 반응 | ★★★☆☆ | 좋아요, 댓글, 공유, 이웃 추가 |
| 원본성 | ★★★☆☆ | 복사/스크래핑 아닌 오리지널 콘텐츠 |

### C-Rank 개선 전략

1. **주제 선택**: 1개 대주제 + 3개 소주제로 블로그 운영
2. **포스팅 빈도**: 주 3-5회, 일정한 시간대
3. **콘텐츠 길이**: 2,000자 이상 + 이미지 15장 이상
4. **내부 링크**: 관련 포스팅 2-3개 상호 연결
5. **댓글 관리**: 24시간 내 답글, 질문형 CTA
6. **이웃 소통**: 이웃 블로거 방문/댓글 일 5-10건

### C-Rank가 이커머스에 미치는 영향

- 높은 C-Rank → 네이버 쇼핑 검색 상위 노출
- 스토어 찜 수 + 리뷰 응답률도 간접 영향
- 블로그 체험단 리뷰가 C-Rank 높은 블로거일수록 노출 효과 극대화

## 2. 네이버 D.I.A.+ (Deep Intent Analysis)

검색 의도와 콘텐츠 매칭 품질을 평가하는 알고리즘.

### 핵심 원리

- 검색어의 **의도(intent)**를 분석하여 가장 적합한 콘텐츠를 상위 노출
- 단순 키워드 매칭이 아닌 **문맥적 관련성** 평가
- 사용자 체류시간, 이탈률, 클릭 후 행동을 반영

### D.I.A.+ 최적화 전략

1. **검색 의도 분류**:
   - 정보형: "OO 사용법", "OO 비교"
   - 상업형: "OO 추천", "OO 가격"
   - 탐색형: "OO 후기", "OO 장단점"
   - 거래형: "OO 구매", "OO 할인"

2. **콘텐츠 구조화**:
   - H2/H3 소제목으로 의도별 섹션 분리
   - 질문-답변 형식으로 FAQ 포함
   - 목차(TOC) 제공으로 탐색 편의

3. **멀티미디어 활용**:
   - 이미지: 직접 촬영/제작 > 무료 이미지
   - 동영상: 30초-3분 설명 영상 삽입
   - 인포그래픽: 비교표, 프로세스 다이어그램

4. **체류시간 최적화**:
   - 글 서두에 "이 글에서 알 수 있는 것" 요약
   - 중간중간 하이라이트 박스로 핵심 강조
   - 글 말미에 관련 포스팅 링크 제공

## 3. 네이버 쇼핑 SEO

### 검색 노출 공식(추정)

```
노출 점수 = 키워드 관련성(30%) + 판매 실적(25%) + 리뷰 점수(20%)
          + 가격 경쟁력(15%) + 스토어 신뢰도(10%)
```

### 핵심 최적화 항목

| 항목 | 방법 | 우선순위 |
|------|------|---------|
| 상품명 | 핵심 키워드 앞배치, 50자 이내 | P0 |
| 카테고리 | 정확한 카테고리 매칭 | P0 |
| 상품 속성 | 모든 필드 100% 입력 | P0 |
| 리뷰 | 포토 리뷰 유도, 응답률 90%+ | P1 |
| 가격 | 최저가 ±10% 이내 유지 | P1 |
| 배송 | 빠른배송/무료배송 설정 | P1 |
| 스토어찜 | 팔로워 수 증가 캠페인 | P2 |

### 금지 행위 (패널티 대상)

- 키워드 스터핑 (상품명에 무관한 키워드 나열)
- 허위 할인 표시 (원가 부풀리기)
- 리뷰 조작 (자가 리뷰, 대가성 리뷰 미표기)
- 이미지 도용 (타 스토어 이미지 무단 사용)
- 카테고리 오분류 (노출 목적 잘못된 카테고리 등록)

## 4. 구글 SEO (한국 시장)

### Core Web Vitals 기준

| 지표 | Good | Needs Improvement | Poor |
|------|------|-------------------|------|
| LCP (Largest Contentful Paint) | ≤ 2.5s | ≤ 4.0s | > 4.0s |
| CLS (Cumulative Layout Shift) | ≤ 0.1 | ≤ 0.25 | > 0.25 |
| INP (Interaction to Next Paint) | ≤ 200ms | ≤ 500ms | > 500ms |

### E-E-A-T 강화 방법

- **Experience**: 실제 사용/체험 기반 콘텐츠 작성
- **Expertise**: 저자 프로필 + 자격증/경력 표시
- **Authoritativeness**: 외부 인용, 백링크 확보
- **Trustworthiness**: HTTPS, 개인정보처리방침, 고객센터 명시

### 구조화 데이터 (JSON-LD)

이커머스 필수 스키마:

```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "제품명",
  "image": "https://example.com/image.jpg",
  "description": "제품 설명",
  "brand": { "@type": "Brand", "name": "브랜드명" },
  "offers": {
    "@type": "Offer",
    "price": "49000",
    "priceCurrency": "KRW",
    "availability": "https://schema.org/InStock"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.8",
    "reviewCount": "1234"
  }
}
```

추가 권장: `FAQPage`, `BreadcrumbList`, `Review`

## 5. GEO (Generative Engine Optimization)

AI 검색 엔진(ChatGPT, Gemini, Perplexity)에서 인용되기 위한 최적화.

### 전략

1. **권위 있는 출처 포지셔닝**: 데이터 기반 주장 + 출처 명시
2. **구조화된 답변 형식**: Q&A, 리스트, 테이블 활용
3. **주제 권위 클러스터**: 피처 페이지 + 서브 토픽 페이지 상호 링크
4. **고유 데이터 생산**: 설문, 사례 연구, 벤치마크 결과
5. **인용 가능한 문장**: "결론적으로 ~이다"식 명확한 서술

### AI 검색 인용률 높이는 콘텐츠 패턴

- 숫자 기반 주장 ("2025년 기준 한국 이커머스 시장 규모 210조원")
- 비교표 (A vs B vs C)
- 단계별 가이드 (Step 1, 2, 3...)
- 정의형 서두 ("X란 ~을 의미하는 개념이다")

## 6. 한국 시장 키워드 전략

### 키워드 리서치 도구

| 도구 | 용도 | 비용 |
|------|------|------|
| 네이버 키워드 플래너 | 네이버 검색량, 경쟁도 | 무료 |
| 네이버 데이터랩 | 트렌드, 연령/성별 분석 | 무료 |
| 블랙키위 | 키워드 발굴, 경쟁 분석 | 유료 |
| 아이템스카우트 | 쇼핑 키워드, 카테고리 분석 | 유료 |
| Google Keyword Planner | 구글 검색량, CPC | 무료 |
| Ahrefs/SEMrush | 백링크, 경쟁사 분석 | 유료 |

### 키워드 분류 체계

```
대분류 키워드 (월 검색량 10,000+)
  └─ 중분류 키워드 (월 1,000-10,000)
       └─ 롱테일 키워드 (월 100-1,000)
            └─ 구매 의도 키워드 ("OO 최저가", "OO 할인 쿠폰")
```

### 시즌별 키워드 캘린더

| 시기 | 키워드 테마 | 비고 |
|------|-----------|------|
| 1월 | 새해 다이어리, 신년 선물 | 설날 전 수요 급증 |
| 2-3월 | 입학/졸업 선물, 봄 패션 | 학기 시작 |
| 5월 | 어버이날, 어린이날 선물 | 가정의 달 |
| 7-8월 | 여름 휴가, 캠핑 용품 | 바캉스 시즌 |
| 9월 | 추석 선물세트 | 명절 대목 |
| 11월 | 블프, 빼빼로데이 | 연말 쇼핑 시작 |
| 12월 | 크리스마스, 연말 정산 | 선물 + 자기 구매 |
