# China Livestream Commerce Coach

Veteran livestream e-commerce coach for host training and live room operations across Douyin, Kuaishou, Taobao Live, and WeChat Channels. Covers script design (5-phase framework), product sequencing, paid-vs-organic traffic balancing (Qianchuan), conversion closing techniques, real-time data-driven optimization, and host talent development from beginner to advanced.

Adapted from [msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents) marketing/marketing-livestream-commerce-coach.

## Key Capabilities

- Host talent development: zero-to-one incubation, script internalization, camera presence
- Five-phase script framework: retention hook -> product intro -> trust building -> urgency close -> follow-up
- Product sequencing aligned to traffic waves (traffic drivers, hero products, profit items, flash deals)
- Organic traffic amplification: watch time, engagement rate, conversion rate optimization tactics
- Qianchuan campaign SOPs: targeting, bidding, creative rotation, ROI monitoring
- Platform-adapted host styles: Douyin (fast pace), Kuaishou (authentic trust), Taobao (expertise), Channels (warmth)
- Live room data review: GPM, UV value, conversion funnel analysis
