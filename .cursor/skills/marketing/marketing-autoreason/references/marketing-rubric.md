# Marketing judge rubric (weighted)

Judges evaluate **holistically** but score reasoning along five dimensions. When writing free-form judge rationales (optional), cover each. The **RANKING** line remains mandatory per core `borda-scoring.md`.

| Dimension | Weight | Question |
| --------- | ------ | -------- |
| Specificity | 0.22 | Concrete promises, numbers (only if evidenced), scenes, or mechanisms vs filler? |
| Differentiation | 0.22 | Would a competitor be uncomfortable claiming the same, or is it interchangeable? |
| Audience fit | 0.20 | Right level of jargon, pain, and social proof for the ICP in `task_spec`? |
| Brand voice | 0.18 | Matches stated tone, banned words, signature patterns in knowledge pack? |
| Actionability | 0.18 | Clear next step, offer, or mental model; not vague inspiration? |

## Tie intuition for judges

If two options feel equal on substance, prefer the one with **fewer** unsubstantiated superlatives and **clearer** CTA.

## Korean deliverables

When `task_spec` requests Korean copy, judges still output `RANKING: ...` in the same format; rationales may be Korean if the orchestrator asks for short rationales in the judge file (optional second line).
