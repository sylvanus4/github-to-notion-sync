# Final Report Template

Generate the report in Korean using the exact structure below. Replace all placeholders with actual data.

## Core Principles

1. **코드에 근거한 지적만** — 추측이나 일반론 금지
2. **발견 시 반드시 `파일명:라인번호` 명시**
3. **`[해당없음]`은 플래그 비활성 시에만** — 코드에 없다는 이유로 남용 금지
4. **잘한 점은 구체적 근거와 함께** — "전반적으로 잘 짰습니다" 같은 공허한 칭찬 금지
5. **모든 출력은 한국어**
6. **Critical / High 발견 항목에는 반드시 diff 형식 수정 제안 포함**

    - **재현 시나리오**: 어떤 상황에서 터지는지
    - **수정 제안**:
    ```diff

    ```
