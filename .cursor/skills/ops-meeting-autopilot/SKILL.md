# ops-meeting-autopilot

## Overview
Pre-meeting: scans open issues from Notion → assembles agenda. During: runs meeting-digest. Post: DMs action items to assignees + updates Notion.

## Autonomy Level
**L3** — AI executes full meeting lifecycle, human reviews outputs

## Skills Used
- meeting-digest, pm-execution, gws-calendar, md-to-notion
