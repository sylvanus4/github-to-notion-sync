# Content Quality Scoring Rubric

Standardized rubric for evaluating platform-native posts generated by `content-graph-produce`.

## Dimensions (5)

Each dimension is scored 1-10. The composite score is the unweighted average.

### 1. Voice Fidelity (VF)

Does the post sound like the brand?

| Score | Criteria |
|-------|----------|
| 9-10 | Indistinguishable from brand's best human-written content; voice DNA fully expressed |
| 7-8 | Clearly on-brand; minor deviations in word choice or rhythm |
| 5-6 | Generally on-brand but feels generic; could be any similar brand |
| 3-4 | Noticeable voice drift; uses vocabulary or tone outside brand guidelines |
| 1-2 | Completely off-brand; violates "never" rules from brand voice DNA |

**Evidence sources**: `voice/brand-voice.md`, `voice/platform-tone.md`

### 2. Platform Native (PN)

Does the post follow the target platform's format and conventions?

| Score | Criteria |
|-------|----------|
| 9-10 | Perfectly formatted; uses all platform-specific features; feels native |
| 7-8 | Correct format; minor missed opportunities (e.g., could use more line breaks) |
| 5-6 | Acceptable format but reads like a cross-posted piece |
| 3-4 | Wrong length, missing platform conventions, or misused features |
| 1-2 | Completely ignores platform rules; would look out of place |

**Evidence sources**: `platforms/{name}.md`

### 3. Hook Strength (HS)

Does the opening line stop the scroll?

| Score | Criteria |
|-------|----------|
| 9-10 | Irresistible hook; creates immediate curiosity, emotion, or urgency |
| 7-8 | Strong hook; clear value proposition in first line |
| 5-6 | Adequate opener; states the topic but doesn't compel |
| 3-4 | Weak or generic opener; "In this post, I'll discuss..." |
| 1-2 | No hook; buries the lead; starts with context instead of value |

**Evidence sources**: `engine/hooks.md`

### 4. Audience Fit (AF)

Does the post resonate with the target audience?

| Score | Criteria |
|-------|----------|
| 9-10 | Directly addresses audience pain points; uses their language; feels personal |
| 7-8 | Relevant to audience; clear value; mostly speaks their language |
| 5-6 | Generally relevant but could target anyone; not segment-specific |
| 3-4 | Misaligned with audience interests or sophistication level |
| 1-2 | Wrong audience entirely; uses jargon they wouldn't know or topics they don't care about |

**Evidence sources**: `audience/segments.md`, `audience/pain-points.md`

### 5. Goal Alignment (GA)

Does the post serve its intended content goal?

| Score | Criteria |
|-------|----------|
| 9-10 | Perfectly serves the goal (awareness/trust/conversion/community); CTA matches goal |
| 7-8 | Clearly serves the goal; CTA present and appropriate |
| 5-6 | Loosely aligned with goal; CTA is weak or missing |
| 3-4 | Goal is unclear from the content; conflicting signals |
| 1-2 | Actively works against the stated goal |

**Evidence sources**: `engine/goals.md`

## Composite Score

```
Composite = (VF + PN + HS + AF + GA) / 5
```

## Grade Bands

| Composite | Grade | Action |
|-----------|-------|--------|
| 9.0-10.0 | A | Ready to publish |
| 7.0-8.9 | B | Minor polish recommended |
| 5.0-6.9 | C | Needs revision; flag specific weak dimensions |
| 3.0-4.9 | D | Significant rework needed; regenerate with adjusted context |
| 1.0-2.9 | F | Discard and regenerate; investigate graph for root cause |

## Score Report Format

```markdown
## Quality Score: {Platform}

| Dimension | Score | Notes |
|-----------|-------|-------|
| Voice Fidelity | {N}/10 | {Brief justification} |
| Platform Native | {N}/10 | {Brief justification} |
| Hook Strength | {N}/10 | {Brief justification} |
| Audience Fit | {N}/10 | {Brief justification} |
| Goal Alignment | {N}/10 | {Brief justification} |
| **Composite** | **{N}/10** | **Grade: {A/B/C/D/F}** |

### Improvement Suggestions
- {Specific, actionable suggestion for the weakest dimension}
- {Second suggestion if composite < 7.0}
```
