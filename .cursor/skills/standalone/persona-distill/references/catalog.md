## 1. Self Distillation & Meta Tools

Skills for distilling yourself, automating the distillation process, or building distillation frameworks.

| Skill | Repository | Description |
|-------|-----------|-------------|
| **Nuwa** | `alchaincyf/nuwa-skill` | The foundational meta-tool for persona distillation. Distills anyone's thinking patterns into a reusable Agent Skill — expressions, mental models, decisions, anti-patterns, and honesty boundaries. |
| **Self-Distillation** | `alchaincyf/self-distillation-skill` | Extracts your own expression style, mental models, and habits from your writings, conversations, and social media. |
| **Distill.skill** | `cursor-ai-downloads/distill.skill` | Fork/derivative of Nuwa for streamlined persona distillation workflows. |
| **Nuwa English** | `cursor-ai-downloads/nuwa-english` | English version of the Nuwa meta-tool. |
| **Auto Distill** | `cursor-ai-downloads/auto-distill-skill` | Automatically distills a persona from provided reference materials with minimal manual input. |
| **Auto Distill v2** | `cursor-ai-downloads/auto-distill-v2` | Enhanced auto-distillation with improved extraction quality. |
| **Digital Immortal** | `cursor-ai-downloads/digital-immortal` | Creates a comprehensive digital twin from a person's complete digital footprint. |
