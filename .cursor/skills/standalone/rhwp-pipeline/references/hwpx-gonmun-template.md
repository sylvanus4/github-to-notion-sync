# HWPX 공문서 양식 (gonmun)

> 행정/비즈니스 공문서 표준 스펙 | Source: modu-ai/cowork-plugins moai-office v1.0.0

대한민국 행정기관 및 기업 공문서 형식을 준수하는 HWPX 템플릿.
`행정 효율과 협업 촉진에 관한 규정` 별지 서식 기반.

---

## 1. 페이지 설정

```yaml
여백:
  위: 35mm    # 발신기관 로고/도장 공간
  아래: 25mm
  왼쪽: 40mm  # 공문서 표준 여백
  오른쪽: 30mm
  머리글: 18mm
  바닥글: 12mm
```

---

## 2. 공문서 구조 (섹션 배치)

```
┌──────────────────────────────────────────┐
│ [머리글] 기관명                           │
├──────────────────────────────────────────┤
│ 수신: {{수신자}}                          │
│ (경유: {{경유기관}})                      │
│                                          │
│ 제목: {{제목}}                            │
│                                          │
│ 1. {{본문내용}}                           │
│                                          │
│ 붙임 1. {{붙임1}} 끝.                     │
│                                          │
│              {{기관명}}장                 │
│                                          │
│ 시행  {{시행기관코드}}-{{문서번호}}        │
│ 접수  {{접수기관코드}}-{{접수번호}}        │
│ 우){{우편번호}} {{주소}}                   │
│ 홈페이지 {{홈페이지URL}}                  │
│ 전화번호 {{전화번호}} 팩스번호 {{팩스}}    │
│ 담당자 {{담당자이메일}}                    │
│ [공개구분] 공개                           │
└──────────────────────────────────────────┘
```

## 3. 필수 섹션 정의

### 3.1 두문 (문서 시작부)

```yaml
두문:
  기관명:
    위치: 중앙 정렬
    폰트: 맑은 고딕 18pt 굵게
  문서분류:
    형식: "{{분류기호}}-{{문서번호}}"
    위치: 우측 상단
    폰트: 10pt
```

### 3.2 수신처 블록

```yaml
수신처:
  레이블: "수신"
  폰트: 10pt 굵게
  형식: |
    수신: {{수신자}} ({{수신담당자}})
    (경유)
  비고: 경유 불필요 시 생략 가능
```

### 3.3 제목

```yaml
제목:
  레이블: "제목"
  폰트: 11pt 굵게
  형식: "제목: {{문서제목}}"
  여백_위: 12pt
  여백_아래: 12pt
```

### 3.4 본문

```yaml
본문:
  들여쓰기: 10mm
  번호체계: "1. -> 가. -> (1) -> (1)"
  줄간격: 160%
  폰트: 10pt
  번호단락:
    1단계: "1. {{내용}}"  # 폰트 굵게
    2단계: "  가. {{내용}}"
    3단계: "    (1) {{내용}}"
    4단계: "      (1) {{내용}}"
```

### 3.5 붙임

```yaml
붙임:
  형식: "붙임  {{번호}}. {{파일명 또는 내용}}  {{매수}}부.  끝."
  조건:
    단일_붙임: "붙임  {{내용}}  1부.  끝."
    복수_붙임: |
      붙임  1. {{붙임1내용}}  1부.
            2. {{붙임2내용}}  1부.  끝.
    붙임_없음: 본문 마지막에 "끝." 마침
```

### 3.6 결문

```yaml
결문:
  발신명의:
    위치: 오른쪽 정렬
    형식: "{{기관명}}장"
    폰트: 11pt 굵게
    여백_위: 20mm
  시행일자_접수:
    형식: |
      시행  {{시행기관코드}}-{{연도}}-{{번호}}  ({{시행일자}})
      접수  {{접수기관코드}}-{{연도}}-{{번호}}  ({{접수일자}})
    폰트: 9pt
  주소줄:
    구분선: 실선 0.2mm
    형식: |
      우) {{우편번호}} {{도로명주소}}
      홈페이지 {{홈페이지URL}}  전화번호 {{전화번호}}  팩스번호 {{팩스번호}}
      공무원 성명 {{담당자명}}  전자우편주소 {{이메일}}
    폰트: 8pt
  공개구분:
    우측: "/공개/"
    폰트: 8pt
```

## 4. 변수 플레이스홀더

| 변수 | 설명 | 필수 |
|------|------|------|
| `{{기관명}}` | 발신 기관 전체명 | Y |
| `{{분류기호}}` | 업무 분류 기호 (예: 총무-123) | Y |
| `{{문서번호}}` | 연도-일련번호 (예: 2026-0042) | Y |
| `{{수신자}}` | 수신 기관/개인 | Y |
| `{{수신담당자}}` | 수신 담당자명 | - |
| `{{경유기관}}` | 경유 기관 (필요 시) | - |
| `{{문서제목}}` | 공문 제목 | Y |
| `{{본문내용}}` | 본문 내용 블록 | Y |
| `{{붙임1}}` | 첫 번째 붙임 파일명 | - |
| `{{시행일자}}` | 시행 날짜 (YYYY. MM. DD.) | Y |
| `{{담당자명}}` | 담당 공무원/직원 이름 | Y |
| `{{이메일}}` | 담당자 이메일 | Y |
| `{{전화번호}}` | 기관 대표 전화 | Y |
| `{{우편번호}}` | 기관 우편번호 | Y |
| `{{도로명주소}}` | 기관 도로명 주소 | Y |

## 5. python-hwpx 연동 예시

```python
from hwpx import HWPXWriter

writer = HWPXWriter()

# 두문
writer.add_paragraph("ThakiCloud", style="Title", align="center")
writer.add_paragraph("총무-2026-0042", align="right", font_size=10)

# 수신
writer.add_paragraph("수신: OO기관장", font_weight="bold", font_size=10)

# 제목
writer.add_paragraph("제목: 2026년 하반기 AI 플랫폼 협업 제안",
                     font_weight="bold", font_size=11)

# 본문
writer.add_paragraph("1. 귀 기관의 무궁한 발전을 기원합니다.",
                     indent=10, line_spacing=160)
writer.add_paragraph("2. ThakiCloud AI Platform 협업 관련 제안입니다.",
                     indent=10, line_spacing=160)

# 붙임
writer.add_paragraph("붙임  1. AI 플랫폼 소개서  1부.  끝.")

# 발신명의
writer.add_paragraph("ThakiCloud 대표이사", align="right",
                     font_weight="bold", font_size=11)

# 시행/접수
writer.add_paragraph("시행  총무-2026-0042  (2026. 05. 06.)",
                     font_size=9)

writer.save("outputs/rhwp/gonmun_sample.hwpx")
```
