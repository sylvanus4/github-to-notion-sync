## 중복 수혜 제한 주의사항

1. **창업패키지 계열 내 중복 불가**: 예비/초기/도약 동시 수혜 불가 (졸업 후 다음 단계 지원 가능)
2. **동일 내용 다기관 중복 제한**: 같은 아이템으로 다른 기관에서 동시에 R&D 자금 수령 불가
3. **TIPS 특수 규정**: TIPS 수혜 중 일부 창업 지원사업 제한 (공고문 확인 필수)
4. **부정 수급 적발 시**: 전액 환수 + 최대 5년간 정부 지원사업 참여 제한
5. **중복 확인 의무**: 신청자가 직접 기존 수혜 이력 신고 (허위 신고 시 제재)

> 구체적 중복 제한 범위는 사업별 공고문에서 반드시 확인하세요.
