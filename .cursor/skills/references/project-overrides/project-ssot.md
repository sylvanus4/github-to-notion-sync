## Database

- PostgreSQL 16 via SQLAlchemy + Alembic
- Migrations: `backend/alembic/versions/`
- Connection: `settings.DATABASE_URL` (uppercase)

## API

- Prefix: `/api/v1/`
- OpenAPI docs: auto-generated by FastAPI
- No manual Swagger files

## Data Flow

```
Yahoo Finance / investing.com / RSS feeds
    ↓
backend/scripts/ (sync, fetch, compute)
    ↓
PostgreSQL (stock_prices, events, patterns, indicators)
    ↓
FastAPI /api/v1/* endpoints
    ↓
React Frontend (TanStack Query → Components → Recharts)
    ↓
outputs/ (DOCX reports, JSON snapshots)
    ↓
Slack (#h-report, #h-daily-stock-check, #효정-할일)
```
