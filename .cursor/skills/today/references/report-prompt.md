# 일간 주식 분석 보고서 — DOCX 생성 프롬프트

이 프롬프트는 `anthropic-docx` 스킬을 사용하여 전문가 수준의 한국어 .docx 주식 분석 보고서를 생성합니다. 종목별 상세 분석(이동평균선, 볼린저 밴드, 오실레이터)과 매매 판단 근거를 포함합니다.

## 시스템 프롬프트

```
전문가 수준의 일간 주식 분석 보고서를 한국어로 작성합니다. docx-js 라이브러리(anthropic-docx 스킬)를 사용하여 .docx 파일을 생성합니다.

설계 원칙:
- 모든 텍스트는 한국어로 작성
- 종목별 상세 분석 포함 (이동평균선 20/55/200일, 볼린저 밴드, 오실레이터 RSI/MACD/Stochastic/ADX)
- 매수/매도 판단 근거를 전문가 관점에서 상세히 서술
- 도표와 분석 텍스트의 균형 — 표만 나열하지 않고 해석을 함께 제공
- 모든 표에 교대 행 음영(#F2F2F2) 적용
- 숫자는 오른쪽 정렬, 텍스트는 왼쪽 정렬
- 통화 표기: 미국 주식 $, 한국 주식 ₩
- 변동률은 +/- 부호와 소수점 2자리 포함

문서 구조:

1. 표지
   - "일간 주식 분석 보고서"
   - 날짜: {date}
   - 부제: "{total_stocks}개 종목 분석 | AlphaEar 생성"

2. 요약 (1페이지)
   - 핵심 발견사항 5~8개 (불릿 포인트)
     예: "이동평균선 정배열 종목 {n}개로 전체 대비 {pct}% — 상승 추세 우세"
     예: "볼린저 밴드 스퀴즈 발생 종목 {n}개 — 변동성 확대 임박"
     예: "골든크로스(20일/55일) 발생 종목: {tickers}"
     예: "RSI 과매수 {n}개, RSI 과매도 {n}개"
     예: "MACD 골든크로스 발생 {n}개 종목"
   - 시그널 분포표:
     | 시그널      | 종목 수 | 비율  |
     | 강력 매수   | n      | n%   |
     | 매수       | n      | n%   |
     | 중립       | n      | n%   |
     | 매도       | n      | n%   |
     | 강력 매도   | n      | n%   |

2.5 시장 동향 (뉴스 데이터가 있는 경우)
   - alphaear-news로 수집한 주요 금융 뉴스 헤드라인 3~5개
   - 각 헤드라인에 출처와 요약 포함
   - 뉴스 데이터가 없는 경우 이 섹션 생략

3. 카테고리별 요약
   | 카테고리 | 종목 수 | 매수 | 중립 | 매도 | 평균 변동률 |
   - 카테고리별 한 행 (AI/반도체, 방산, 에너지, 바이오 등)
   - 매수 신호가 가장 많은 카테고리 굵게 표시

4. 주요 종목 상세 분석 — BUY/SELL 종목 (핵심 섹션)
   BUY/STRONG_BUY 또는 SELL/STRONG_SELL 시그널 종목 각각에 대해:

   ■ {종목명} ({티커}) — {종합시그널}

   가격 동향:
   | 항목     | 값          |
   | 현재가   | {price}     |
   | 변동률   | {change}%   |
   | 거래량   | {volume}    |

   이동평균선 분석:
   | 지표      | 값        | 가격 대비    |
   | 20일 SMA  | {sma20}  | {above/below} |
   | 55일 SMA  | {sma55}  | {above/below} |
   | 200일 SMA | {sma200} | {above/below} |
   | 배열 상태  | {정배열/역배열/혼조} |      |

   이동평균선 해석 (프로그래밍 방식으로 생성):
   - 정배열인 경우: "20일, 55일, 200일 이동평균선이 정배열(20>55>200) 상태로 강한 상승 추세를 나타내고 있습니다. 단기·중기·장기 추세가 모두 상승 방향으로 정렬되어 있어 추세 추종 매수에 유리한 환경입니다."
   - 역배열인 경우: "20일, 55일, 200일 이동평균선이 역배열(20<55<200) 상태로 하락 추세가 지속되고 있습니다. 반등 시도가 있더라도 이평선 저항에 막힐 가능성이 높습니다."
   - 혼조인 경우: "이동평균선이 혼조 상태로 추세 전환 과정에 있습니다. 방향성 확인 후 진입하는 것이 바람직합니다."
   - 가격 > SMA200: "현재 가격이 200일 이동평균선 위에 위치하여 장기 상승 추세를 유지하고 있습니다."
   - 가격 < SMA200: "현재 가격이 200일 이동평균선 아래에 위치하여 장기 하락 추세에 있습니다. 200일선 회복 여부가 핵심 관건입니다."
   - 골든크로스 발생: "최근 20일선이 55일선을 상향 돌파하여 골든크로스가 발생했습니다. 중기 추세 전환 신호로 매수 관점에서 긍정적입니다."
   - 데스크로스 발생: "최근 20일선이 55일선을 하향 돌파하여 데스크로스가 발생했습니다. 중기 하락 전환 신호로 주의가 필요합니다."

   볼린저 밴드 분석:
   | 지표        | 값           |
   | 상단 밴드   | {upper}      |
   | 중간선      | {middle}     |
   | 하단 밴드   | {lower}      |
   | %B         | {pct_b}      |
   | 밴드폭      | {bandwidth}  |
   | 스퀴즈      | {yes/no}     |

   볼린저 밴드 해석 (프로그래밍 방식으로 생성):
   - %B > 1.0: "현재 가격이 볼린저 밴드 상단을 돌파하여 강한 상승 모멘텀을 보이고 있습니다. 다만 과매수 구간으로 단기 조정 가능성도 존재합니다."
   - %B > 0.8: "현재 가격이 볼린저 밴드 상단에 근접하여 과매수 구간에 있습니다. 추세 강도가 유지되는지 확인이 필요합니다."
   - %B < 0.0: "현재 가격이 볼린저 밴드 하단을 이탈하여 과매도 구간에 있습니다. 평균 회귀 관점에서 반등 가능성이 있으나 하락 추세 지속 여부를 확인해야 합니다."
   - %B < 0.2: "현재 가격이 볼린저 밴드 하단에 근접하여 지지선 부근입니다."
   - 스퀴즈 발생: "밴드폭이 최근 20일 최저치로 축소되어 스퀴즈 상태입니다. 조만간 큰 변동성 확대가 예상되며, 돌파 방향에 주목해야 합니다."

   오실레이터 분석:
   | 지표              | 값          | 판단             |
   | RSI(14)          | {rsi14}     | {과매수/과매도/중립} |
   | MACD             | {macd_line} | {골든크로스/데드크로스/양수/음수} |
   | MACD 히스토그램   | {histogram} |                   |
   | 스토캐스틱 %K/%D | {k}/{d}     | {과매수/과매도/중립} |
   | ADX(14)          | {adx}       | {강한추세/보통추세/추세없음} |
   | 오실레이터 종합   |             | {매수/중립/매도}   |

   오실레이터 해석 (프로그래밍 방식으로 생성):
   - RSI > 70: "RSI(14)가 {value}로 과매수 구간(>70)에 있습니다. 단기 조정 가능성이 높으며, 이익 실현을 고려할 시점입니다."
   - RSI < 30: "RSI(14)가 {value}로 과매도 구간(<30)에 있습니다. 반등 가능성에 주목할 필요가 있습니다."
   - MACD 골든크로스: "MACD 골든크로스가 발생하여 상승 모멘텀이 강화되고 있습니다."
   - MACD 데드크로스: "MACD 데드크로스가 발생하여 하락 모멘텀이 강화되고 있습니다."
   - ADX ≥ 25: "ADX가 {value}로 추세 강도가 강한 상태입니다. 현재 추세 방향으로의 포지션 유지가 유리합니다."
   - ADX < 15: "ADX가 {value}로 추세가 약한 상태입니다. 횡보 구간으로 추세 추종 전략은 불리할 수 있습니다."

   뉴스 감성 (감성 데이터가 있는 경우):
   "뉴스 감성: {긍정/부정/중립}({score}) — {summary}"
   - 감성 데이터가 없는 경우 이 행 생략

   종합 매매 판단:
   "{종합시그널}" — {매수/매도 근거를 이동평균선, 볼린저 밴드, 오실레이터 분석을 종합하여 2~3문장으로 서술}

5. 전체 종목 시그널 일람표
   | 티커 | 종목명 | 현재가 | 변동률 | 터틀 | 볼린저 | 오실레이터 | 종합 판단 |
   - 종합 시그널 강도순 정렬 (강력 매수 → 강력 매도)

6. 핫 종목 — 미추적 종목 (발견 데이터가 있는 경우)
   | 지수 | 티커 | 종목명 | 카테고리 | 현재가 | 변동률 | 거래량 | 거래대금 |
   - NASDAQ 100, KOSPI 100, KOSDAQ 100 각 1종목
   - 각 종목이 "핫"한 이유 간략 설명

7. 기술적 지표 상세표
   7a. 이동평균선 상세:
   | 티커 | SMA20 | SMA55 | SMA200 | 배열 | 돈치안 상단 | 돈치안 하단 | ATR |
   - BUY 또는 SELL 시그널 종목만 표시

   7b. 볼린저 밴드 상세:
   | 티커 | 상단 | 중간선 | 하단 | %B | 밴드폭 | 스퀴즈 |
   - BUY 또는 SELL 시그널 종목만 표시

   7c. 오실레이터 상세:
   | 티커 | RSI(14) | RSI 구간 | MACD | 히스토그램 | Stoch %K/%D | ADX | 오실 판단 |
   - BUY 또는 SELL 시그널 종목만 표시

8. 리스크 노트
   - 주요 리스크 3~4개 불릿
   - "오실레이터(RSI, MACD, 스토캐스틱, ADX)는 보조 확인 지표로 활용됩니다. 이동평균선/볼린저 밴드와 함께 3중 확인 체계로 종합 판단합니다."
   - 면책 조항: "본 보고서는 정보 제공 목적으로 작성되었으며 투자 권유를 목적으로 하지 않습니다. 투자의 최종 판단과 책임은 투자자 본인에게 있습니다."

서식 규칙 (docx-js):
- 페이지: US Letter (12240 × 15840 DXA)
- 여백: 1인치 전방향 (1440 DXA)
- 본문 글꼴: Arial 11pt, 제목1 16pt 굵게, 제목2 13pt 굵게
- 표 테두리: 가는 실선 (BorderStyle.SINGLE, size: 1)
- 헤더 행: 굵게, 진회색 배경 (#D9D9D9)
- 교대 행: 흰색 / 연회색 (#F2F2F2)
- 표 너비: 콘텐츠 너비 100% (WidthType.PERCENTAGE, 100)
- 숫자 열: 오른쪽 정렬 (AlignmentType.RIGHT)
- 바닥글: 가운데 페이지 번호, 오른쪽 "대외비"
```

## 데이터 입력 계약

프롬프트는 최대 세 개의 JSON 입력을 기대합니다:

1. **분석 JSON** (`daily_stock_check.py --source db` 출력):
   ```json
   {
     "date": "2026-03-07",
     "total_stocks": 52,
     "results": [{
       "ticker": "NVDA",
       "name": "NVIDIA",
       "category": "ai_semiconductor",
       "price": 285.5,
       "change_pct": 2.15,
       "turtle": {
         "sma20": 280.0, "sma55": 265.0, "sma200": 240.0,
         "price_vs_sma20": "ABOVE", "price_vs_sma55": "ABOVE", "price_vs_sma200": "ABOVE",
         "ma_alignment": "정배열",
         "golden_cross_20_55": false, "death_cross_20_55": false,
         "signal": "BUY", "rationale": "..."
       },
       "bollinger": {
         "upper": 295.0, "middle": 280.0, "lower": 265.0,
         "pct_b": 0.68, "bandwidth": 10.71, "is_squeeze": false,
         "signal": "NEUTRAL", "rationale": "..."
       },
       "oscillators": {
         "rsi14": 55.2, "rsi_zone": "중립",
         "macd_line": 2.35, "macd_signal_line": 1.8, "macd_histogram": 0.55, "macd_cross": null,
         "stoch_k": 65.3, "stoch_d": 62.1, "stoch_zone": "중립",
         "adx": 28.5, "adx_strength": "강한추세",
         "signal": "NEUTRAL", "rationale": "..."
       },
       "overall_signal": "BUY",
       "sentiment": { "score": 0.72, "summary": "최근 실적 발표 호조" }
     }],
     "summary": { "strong_buy": 2, "buy": 8, "neutral": 35, "sell": 5, "strong_sell": 2 }
   }
   ```

2. **발견 JSON** (`discover_hot_stocks.py`, 선택사항):
   ```json
   {
     "date": "2026-03-07",
     "discoveries": [{ "index": "NASDAQ 100", "ticker": "MSTR", ... }]
   }
   ```

3. **뉴스 JSON** (`alphaear-news` 출력, 선택사항):
   ```json
   {
     "headlines": [
       { "title": "...", "source": "...", "summary": "..." }
     ]
   }
   ```

## 출력

`outputs/reports/daily-{date}.docx`에 저장되는 한국어 .docx 파일.
