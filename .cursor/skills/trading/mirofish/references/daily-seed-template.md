# Daily Seed Document Template

Standard template for generating MiroFish seed documents from daily stock analysis outputs.

## Source Files

| File | Content |
|------|---------|
| `outputs/analysis-{date}.json` | Technical signals (Turtle, Bollinger, RSI, MACD, ADX) for all tracked stocks |
| `outputs/news-{date}.json` | Top 5 financial news headlines with summaries |
| `outputs/screener-{date}.json` | Multi-factor screening results |

## Seed Document Structure

```markdown
# Global Market Analysis: {Brief Scenario Title} ({date})

## Executive Summary
2-3 sentences summarizing today's market condition from the news headlines.

## Key Events
### 1. {Event from headline 1}
Paragraph with details, impact, and data points.

### 2. {Event from headline 2}
...
(Include all 5 headlines as sections)

## Current Stock Signals ({date})

### Strong Buy Signals
- **{Name} ({ticker})**: {price} ({change_pct}%), RSI {rsi}, ADX {adx}, Turtle {signal}
(List all stocks with overall_signal == "BUY")

### Strong Sell Signals
- **{Name} ({ticker})**: {price} ({change_pct}%), RSI {rsi}, ADX {adx}, Turtle {signal}
(List all stocks with overall_signal == "SELL")

### Key Neutral — Watch for Breakout
- **{Name} ({ticker})**: {price} ({change_pct}%), RSI {rsi} — {brief context}
(Top 3-5 notable NEUTRAL stocks)

## Stakeholder Groups
Always include these 6 groups for consistency:

### Institutional Investors (Hedge Funds, Pension Funds)
Large allocators managing $10B+ portfolios...

### Retail Investors (Individual Traders)
Self-directed investors on Robinhood, Korean securities apps...

### Federal Reserve Officials
FOMC members balancing inflation, employment, geopolitical risk...

### Corporate Executives (Big Tech)
CEOs of NVIDIA, Meta, Microsoft, Apple, Amazon...

### Energy & Commodities Traders
Traders focused on crude oil, gold, and defense stocks...

### Korean Market Participants (Domestic Institutions & Retail)
Korean investors navigating KOSPI...

## Simulation Objective
Predict how these six stakeholder groups interact over the next 20 trading days...
(Tailor 4-6 key questions to today's specific events)
```

## Python Helper: Auto-Generate Seed

```python
import json
from datetime import date

def generate_seed(analysis_path: str, news_path: str, output_path: str):
    with open(analysis_path) as f:
        analysis = json.load(f)
    with open(news_path) as f:
        news = json.load(f)

    buys = [s for s in analysis['results'] if s['overall_signal'] == 'BUY']
    sells = [s for s in analysis['results'] if s['overall_signal'] == 'SELL']
    neutrals = [s for s in analysis['results'] if s['overall_signal'] == 'NEUTRAL']

    lines = [f"# Global Market Analysis ({analysis['date']})\n"]
    lines.append("## Executive Summary\n")
    lines.append("## Key Events\n")
    for i, h in enumerate(news.get('headlines', []), 1):
        lines.append(f"### {i}. {h['title']}")
        lines.append(f"Source: {h['source']}")
        lines.append(f"{h['summary']}\n")

    for label, stocks in [("Strong Buy Signals", buys), ("Strong Sell Signals", sells)]:
        lines.append(f"## {label}\n")
        for s in stocks:
            osc = s.get('oscillators', {})
            lines.append(f"- **{s['name']} ({s['ticker']})**: {s['price']:,.0f} "
                         f"({s['change_pct']:+.2f}%), RSI {osc.get('rsi14',0):.1f}, "
                         f"ADX {osc.get('adx',0):.1f}, Turtle {s['turtle']['signal']}")
        lines.append("")

    with open(output_path, 'w') as f:
        f.write('\n'.join(lines))
```
