# Latest Data Point Trend Analysis Checklist

Use this checklist **before** stating 8MA slope or BUY status. Prevents misreading recent trend changes.

## Rightmost Data Points

- [ ] Trace the rightmost 3–5 data points on the 8MA (orange) line
- [ ] Determine slope from the **latest** points only, not historical movement
- [ ] Record: rising, falling, or flat based on the rightmost edge

## Failed-Reversal Detection

- [ ] If a trough (purple ▼) was identified: did the subsequent rise sustain?
- [ ] Check if 8MA rose 2–3 bars then fell again (failed reversal)
- [ ] If failed reversal: classify as FAILED; do **not** issue BUY signal
- [ ] Recommend WAIT until a confirmed reversal with sustained rise

## Before Stating BUY Status

- [ ] 8MA slope is rising (from rightmost 3–5 points)
- [ ] Reversal from trough is sustained, not failed
- [ ] At least 2–3 consecutive rises from trough
- [ ] All of the above confirmed before declaring CONFIRMED BUY
