# Autoimprove: trading-market-environment-analysis

Evals: structured output, ≥3 numerics, actionable conclusion, risk/invalidation, no hallucinated data. Tests: global environment, US/Asia/Europe, FX/commodities, catalyst week, data-sparse session.

- **Score**: 20/25 (80%)
- **Weaknesses**: Narrative could quote “markets rose” without sourced levels (EVAL 2/5). Final playbook sometimes vague (EVAL 3). Invalidation rarely explicit (EVAL 4).

- **Score**: 24/25 (96%)
- **Outcome**: Kept — ≥95% target met (single residual edge case: true N/A weeks still need disciplined wording).
