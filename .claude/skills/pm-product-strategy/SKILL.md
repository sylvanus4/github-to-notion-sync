description: Product strategy workflows — vision, Lean Canvas, Business Model Canvas, SWOT, PESTLE, Porter's Five Forces, Ansoff Matrix, and pricing strategy.
arguments: [framework, topic]
---

Run product strategy framework `$framework` on `$topic`.

## Frameworks

| Framework | Use Case |
|-----------|----------|
| vision | Product vision and mission statement |
| lean-canvas | 1-page startup business model |
| bmc | Full Business Model Canvas |
| swot | Strengths, Weaknesses, Opportunities, Threats |
| pestle | Political, Economic, Social, Tech, Legal, Environmental |
| porter | Porter's Five Forces competitive analysis |
| ansoff | Growth strategy matrix (market/product) |
| pricing | Monetization and pricing strategy |
| value-prop | Value proposition design |
| trade-offs | Strategic trade-off analysis |

## Output Format

Each framework produces a structured Korean document with:
1. Framework-specific analysis sections
2. Key insights and implications
3. Strategic recommendations
4. Open questions for stakeholders

## Rules

- Apply Karpathy Opposite Direction Test on all analyses
- Include at least one counter-argument per recommendation
- Cite evidence for market claims where possible
