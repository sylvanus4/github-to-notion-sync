# Game Studio Plugin

Complete game development toolkit for Cursor — provides specialized AI agents, workflow skills, validation hooks, and coding rules for building browser-based games.

## What's Included

### Agents (5 specialists)
- **creative-director** — Vision, theme coherence, player experience
- **game-designer** — Mechanics, balance, progression systems
- **lead-programmer** — Architecture, engine code, performance
- **ui-programmer** — React components, animations, responsive UI
- **qa-tester** — Test plans, edge cases, solvability validation

### Skills (7 workflows)
- **start** — Scaffold a new game project (React + Vite + TypeScript + Tailwind)
- **brainstorm** — Structured ideation with diverge/converge/refine phases
- **design-system** — Game Design Document (GDD) generation
- **dev-story** — Feature decomposition into dev stories with acceptance criteria
- **qa-plan** — Test plan generation with risk-based prioritization
- **balance-check** — Numerical balance analysis for game parameters
- **playtest-report** — Simulated playtesting with 4 player archetypes

### Hooks
- **validate-game-logic.sh** — Pre-commit validation for puzzle bank integrity, type safety, and state consistency

### Rules
- **game-code.mdc** — Pure-function engine code, deterministic PRNG, type contracts, state machine rules
- **game-components.mdc** — React component architecture, Zustand patterns, Tailwind styling, accessibility, responsive design

## Quick Start

1. Invoke the `start` skill to scaffold a new game project
2. Use `brainstorm` to ideate your game concept
3. Run `design-system` to produce a Game Design Document
4. Break features into stories with `dev-story`
5. Build, test, and iterate using the agents and remaining skills

## Target Stack

React 18+ · Vite · TypeScript · Tailwind CSS · Zustand · Canvas/WebGL (optional)
