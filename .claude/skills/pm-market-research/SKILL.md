---
name: pm-market-research
description: Market research workflows — personas, segmentation, journey maps, competitive analysis, sentiment analysis, and market sizing (TAM/SAM/SOM).
arguments: [workflow, topic]
---

Run market research workflow `$workflow` for `$topic`.

## Workflows

| Workflow | Description |
|----------|-------------|
| persona | User persona creation with data-backed attributes |
| segmentation | Market segmentation analysis |
| journey-map | Customer journey mapping with touchpoints |
| competitor | Competitive landscape analysis |
| sentiment | Brand/product sentiment analysis |
| market-size | TAM/SAM/SOM estimation with methodology |

## Output

Structured Korean document with:
1. Research methodology and sources
2. Analysis deliverable (persona card, journey map, etc.)
3. Key insights and implications
4. Confidence levels and data gaps

## Rules

- Cite sources for market size estimates
- Include sensitivity analysis for financial projections
- Clearly label assumptions vs validated data
