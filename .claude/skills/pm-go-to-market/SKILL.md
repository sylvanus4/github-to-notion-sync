Run go-to-market workflow `$workflow` for `$topic`.

## Workflows

| Workflow | Description |
|----------|-------------|
| gtm-strategy | Comprehensive go-to-market plan |
| icp | Ideal Customer Profile definition |
| beachhead | Beachhead segment selection |
| growth-loops | Design viral/content/paid growth loops |
| gtm-motions | Assess PLG vs SLG vs hybrid motions |
| battlecard | Competitive positioning battlecard |
| launch-plan | Product launch checklist and timeline |

## Output

Structured Korean document with:
1. Strategy-specific deliverable
2. TAM/SAM/SOM estimates where applicable
3. Channel recommendations
4. Success metrics and tracking plan

## Rules

- Ground ICP in observable behaviors, not demographics alone
- Include at least 2 competing GTM motion evaluations
- Battlecards must include "why they win" honestly
